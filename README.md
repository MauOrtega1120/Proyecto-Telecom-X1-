# 📡 Telecom X: Customer Churn Analysis

Este proyecto aborda uno de los mayores desafíos en la industria de las telecomunicaciones: la **pérdida de clientes (Churn)**. A través de un pipeline de datos completo, transformé un dataset crudo en un informe estratégico de negocio.

## 🎯 Objetivo
Identificar las variables críticas que influyen en la decisión de los clientes de abandonar la empresa y proporcionar recomendaciones accionables para reducir la tasa de evasión.

## 🛠️ Tecnologías Utilizadas
* **Python 3.x**
* **Pandas:** Limpieza de datos y manipulación de JSON anidados.
* **NumPy:** Operaciones matemáticas y tratamiento de valores nulos.
* **Matplotlib & Seaborn:** Visualización de datos y análisis estadístico.
* **Google Colab:** Entorno de desarrollo.

## 📈 Flujo del Proyecto
1. **Extracción:** Carga y normalización de archivos JSON complejos.
2. **Transformación:** Limpieza de tipos de datos, traducción de variables y creación de nuevas métricas (Cuentas Diarias).
3. **Análisis (EDA):** Identificación de hallazgos clave como la vulnerabilidad en contratos mensuales y el impacto de la tecnología de Fibra Óptica.

## 💡 Hallazgos Clave
* Los clientes con contratos **mes a mes** representan el mayor riesgo de fuga.
* El periodo de mayor abandono ocurre antes de los **12 meses** de permanencia.
* Existe una correlación directa entre cargos mensuales elevados y la decisión de cancelar el servicio.
